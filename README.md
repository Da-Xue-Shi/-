焦氏易林解卦系统 (Jiao Shi Yi Lin Scopes)
这是一个基于 HTML/JS 开发的纯静态解卦工具。收录了《焦氏易林》全书四千零九十六卦的卦辞与林辞，旨在提供一个极简、快速且离线的易学研究工具。

🌟 项目亮点
纯静态架构：无需服务器、无需数据库、无需安装环境。下载后双击 index.html 即可运行。

全量收录：包含 64 卦及其对应的各卦变动（共 4096 条林辞）。

极速搜索：支持按本卦、变卦快速检索对应的易林辞。

响应式设计：完美适配手机、平板及电脑浏览器。

零外部依赖：核心逻辑不依赖 API，断网环境下也能正常解卦。

Bash

2. 运行
进入文件夹，找到 index.html 文件，直接用浏览器（Chrome, Edge, Safari 等）打开即可开始解卦。

📖 使用指南
输入本卦：选择或输入你占得的原始卦象。

输入变卦：选择卦象变动后的结果（若无变卦，则默认为本卦）。

查看结果：系统会自动提取《焦氏易林》中对应的林辞，并展示解释。
